# Documentación de la API

## Endpoints

### Usuarios

#### GET /api/users
Obtiene una lista de usuarios.

#### POST /api/users
Crea un nuevo usuario.

#### PUT /api/users/:id
Actualiza un usuario existente.

#### DELETE /api/users/:id
Elimina un usuario.

## Autenticación

#### POST /api/auth/login
Inicia sesión y retorna un token JWT.

#### POST /api/auth/register
Registra un nuevo usuario.
